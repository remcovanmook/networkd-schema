MTU (Bytes) MTUBytesType

Maximum Transmission Unit (MTU) in bytes

Structure: Integer (68...65535) OR String matching regular expression ^[0-9]+(\s*[KMGTPE]i?)?$ format bytes

Examples:
  • 1500
  • 9000

IPv6 MTU (Bytes) MTUv6BytesType

Maximum Transmission Unit (MTU) in bytes

Structure: Integer (1280...65535) OR String matching regular expression ^[0-9]+(\s*[KMGTPE]i?)?$

Examples:
  • 1280
  • 1500

boolean boolean

A boolean value (true or false).

Structure: Boolean

Size in Bytes bytesType

Size in bytes (Integer or String with suffix B, K, M, G, T, P, E)

Structure: Integer (min: 0) OR String matching regular expression ^[0-9]+(\s*[KMGTPE]i?B?)?$

Examples:
  • 1G
  • 512M
  • 1024

enum enum

A value chosen from a specific set of allowed strings.

Structure: Enum

File Path filenameType

Filesystem path

Structure: String format uri-reference

Examples:
  • /etc/issue

Shell Globs globsType

A whitespace-separated list of shell-style globs.

Structure: String matching regular expression ^(!\s*)?\S+(\s+\S+)*$

Examples:
  • eth*
  • en* wlan*
  • !virbr*
  • !eth0 eth1

integer integer

A whole number.

Structure: Integer

ip_addressType ip_addressType

IPv4 or IPv6 Address

Structure: IPv4 Address OR IPv6 Address

ip_prefixType ip_prefixType

IPv4 or IPv6 Prefix (CIDR)

Structure: IPv4 CIDR OR IPv6 CIDR

IPv4 Address ipv4_addressType

IPv4 Address

Structure: String format ipv4

Examples:
  • 192.168.1.1

IPv4 CIDR ipv4_prefixType

IPv4 Address with Prefix Length (CIDR), e.g., 192.168.1.1/24

Structure: String matching regular expression ^((25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\/(3[0-2]|[1-2]?[0-9]|[0-9])$

Examples:
  • 192.168.1.0/24

IPv6 Address ipv6_addressType

IPv6 Address

Structure: String format ipv6

Examples:
  • 2001:db8::1

IPv6 CIDR ipv6_prefixType

IPv6 Address with Prefix Length (CIDR), e.g., 2001:db8::1/64

Structure: String matching regular expression ^([0-9a-fA-F]{1,4}:){1,7}:?([0-9a-fA-F]{1,4}:?)*\/(12[0-8]|1[0-1][0-9]|[1-9]?[0-9]|[0-9])$

Examples:
  • 2001:db8::/64

MAC Address mac_addressType

MAC Address (Hex separated by colons or hyphens)

Structure: String matching regular expression ^([0-9a-fA-F]{2}[:-]){5}([0-9a-fA-F]{2})$

Examples:
  • 00:11:22:33:44:55

Percentage percentageType

Percentage value (e.g. 10%, 0.5%)

Structure: String matching regular expression ^[0-9]+(\.[0-9]+)?%$ format percent

Examples:
  • 10%
  • 0.5%

QDisc Handle qdisc_handleType

Queueing Discipline handle (hex major:minor).

Structure: String matching regular expression ^[0-9a-fA-F]{1,4}:([0-9a-fA-F]{1,4})?$

Examples:
  • 1:0
  • 10:

QDisc Parent qdisc_parentType

Queueing Discipline parent.

Structure: Enum: root, clsact, ingress, egress OR QDisc Handle

Examples:
  • root
  • ingress
  • 1:1

Data Rate rateType

Data rate in bits per second (e.g. 1G, 100M, 500K)

Structure: Integer (min: 0) OR String matching regular expression ^[0-9]+(\.[0-9]+)?(\s*[KMGTPE]i?)?$

Examples:
  • 1G
  • 100M

Duration secondsType

Time duration (e.g. 5s, 1min, 500ms)

Structure: String matching regular expression ^[0-9]+(\.[0-9]+)?(us|ms|s|min|h|d|w|M|y)?$ format duration

Examples:
  • 5s
  • 100ms
  • 1min

string string

A sequence of characters.

Structure: String
